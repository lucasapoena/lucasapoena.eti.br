# 🧠 Lucas Apoena — Portfolio

Este repositório contém o código-fonte do meu portfólio pessoal. [Visite o site](https://lucasapoena.eti.br)

Sou **Lucas Apoena**, Desenvolvedor Sênior .NET (Full-Stack), com foco em modernização de sistemas, arquitetura evolutiva e aplicações práticas de Inteligência Artificial.

O objetivo deste projeto é:

> Apresentar quem eu sou como profissional, minha base técnica, as tecnologias com as quais trabalho e a forma como penso arquitetura, engenharia e evolução de sistemas.

---

## 🚀 Stack & Abordagem Técnica

Este site foi construído de forma intencionalmente enxuta.

- [x] Sem frameworks pesados.  
- [x] Sem abstrações desnecessárias.  
- [x] Com foco em clareza, controle e performance.

### 🧱 Base Tecnológica

- **HTML5 & CSS3 puro**  
  Layout, Design System próprio, responsividade e animações sem dependências externas.

- **Vanilla JavaScript**  
  Toda a lógica de interação, manipulação do DOM e comportamentos avançados foi implementada sem React/Vue ou bibliotecas auxiliares.

- **Internacionalização dinâmica (i18n)**  
  Suporte nativo a:
  - 🇧🇷 Português
  - 🇺🇸 Inglês
  - 🇪🇸 Espanhol

- **Design inspirado em arquitetura de sistemas**  
  Estética “deep-slate”, blueprint grid, elementos reativos e uma identidade visual alinhada com engenharia e sistemas distribuídos.
